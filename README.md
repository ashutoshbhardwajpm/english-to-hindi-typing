# English to Hindi Typing

A simple **GUI-based Python application** that converts English typed words into Hindi in real-time. As you type in English and press **space**, the last word is automatically transliterated to Hindi.

## Features

- Real-time English to Hindi transliteration
- Simple and clean GUI using Tkinter
- Converts each word as soon as space is pressed
- Lightweight and easy to use
